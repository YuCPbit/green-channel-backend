# 团队拉取与启动验收

## 后端

```bash
git clone https://github.com/YuCPbit/green-channel-backend.git
cd green-channel-backend
mysql -u root -p < docs/数据库设计.sql
mysql -u root -p < docs/数据库初始化数据.sql
mvn test
mvn spring-boot:run
```

按本机环境在 `src/main/resources/application.yml` 或环境变量中配置 MySQL，禁止把真实密码提交到 Git。启动后访问 `GET http://localhost:8080/api/health`，应返回成功响应。

## 前端

```bash
git clone https://github.com/YuCPbit/green-channel-frontend.git
cd green-channel-frontend
npm install
npm run build
npm run dev
```

浏览器打开终端输出的本地地址，确认登录页可以显示，并能调用后端登录接口。

## B/C/D 验收回报

每位成员在群里回复以下内容：

- 成员代号和操作系统；
- Java、Maven、Node、npm、MySQL 版本；
- 后端 `mvn test` 结果；
- `/api/health` 成功截图；
- 前端登录页截图；
- 若失败，附去除账号、密码、Token 后的错误日志。
